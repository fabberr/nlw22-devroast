# Project Context

DevRoast - code scoring platform with optional roast mode.

## Commands
- `pnpm lint` - Linting with Biome
- `pnpm format` - Formatting with Biome
- `pnpm dev` - Start Next.js dev server
- `pnpm build` - Next.js build

## Design
- Use semantic HTML (`<main>`, `<section>`, `<header>`, `<nav>`)
- Follow Pencil design
- Use canonical Tailwind classes
