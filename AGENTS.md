# Project Context

devroast (brand is all lowercase) - a code scoring tool with optional roast mode.

## Commands
- `pnpm lint` - Linting with Biome
- `pnpm format` - Formatting with Biome
- `pnpm dev` - Start Next.js dev server
- `pnpm build` - Next.js build

## General Guidelines
- Use semantic HTML (e.g.: `<main>`, `<section>`, `<header>`, `<nav>`)
- Use Context7 MCP for searching documentation

## Design
- Follow Pencil design (use Pencil MCP)
- Use canonical Tailwind v4 classes
